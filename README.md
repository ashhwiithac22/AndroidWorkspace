# Experiment 1 – Basic UI Driven Android Application 🎨

* A simple Android application built using **GUI components**, **font**, and **colour** settings.
* The app accepts **username and password**, and allows the user to **change background colour** and **font size** using buttons.

## Features

* **User Input Fields**: Enter username and password.
* **Background Colour Change**: Change the screen colour with a button click.
* **Font Size Change**: Increase the text size using a button.
* **Basic Android UI Components**: Buttons, TextView, EditText.

## Steps Followed

1. Open Android Studio and create a new Android project.
2. Choose Android version, package name, and click **Finish**.
3. Navigate to the project in the Package Explorer.
4. Open the `res` folder and select the `layout` directory.
5. Open `activity_main.xml` to design the UI.
6. Add the required UI components in XML.
7. Open the `src` folder and edit `MainActivity.java`.
8. Implement button functions for colour and font changes.
9. Run the project using **Run As → Run Configuration**.
10. View the output on the emulator or device.

# Experiment 2 – Android Services for Travel Agency App 📱

* An Android application for a travel agency implementing **four Android services** to attract visitors to destinations.
* Includes Location Service, Notification Service, Media Playback Service, and Geofencing Service.

## Features

* **Location Service**: Tracks/simulates user location and displays nearby destinations or travel deals.
* **Notification Service**: Sends travel offers, reminders, and promotional messages.
* **Media Playback Service**: Plays travel audio guides or promotional clips in the background.
* **Geofencing Service**: Notifies users when they are near a destination or travel hotspot.

# Ex3 – Data Management Using SQLite3

## 📌 Aim
To develop a mobile application using **Android Studio** and **SQLite3** that performs basic data management operations such as **inserting, deleting, viewing, and editing data**.

---

## 📝 Description
This Android application demonstrates the use of **SQLite3 database** for storing and managing data locally on a mobile device.  
The application provides a user-friendly interface that allows users to add new records, view stored data, update existing records, and delete unwanted entries.

---

## 🛠️ Technologies Used
- **Programming Language:** Java  
- **IDE:** Android Studio  
- **Database:** SQLite3  
- **UI Design:** XML Layouts  

---

## ⚙️ Functionalities Implemented

### 1️⃣ Data Insertion
- Users can enter data through input fields
- Data is stored securely in SQLite database
- Validation is applied to prevent empty inputs

### 2️⃣ Data Deletion
- Users can delete selected records
- Deleted data is removed permanently from the database

### 3️⃣ Data Viewing
- All stored records are displayed in the app interface
- Data is retrieved dynamically from SQLite database

### 4️⃣ Data Editing
- Users can modify existing records
- Updated data is saved back into the database

---

## 🗂️ Database Details
- SQLite database is created locally within the application
- CRUD operations are implemented using SQL queries
- Database helper class is used to manage database creation and version control

---

## 📱 User Interface
- Simple and clean layout
- Input fields for data entry
- Buttons for Insert, View, Update, and Delete operations
- Easy navigation for better user experience

# Experiment 4 – Simple Calculator App 🧮

- An Android application implementing a basic calculator to perform arithmetic operations.
- Includes Addition, Subtraction, Multiplication, and Division functionalities.

## Features

- Addition: Adds two user-input numbers and displays the result.
- Subtraction: Subtracts the second number from the first and displays the result.
- Multiplication: Multiplies two numbers and shows the output.
- Division: Divides the first number by the second and displays the quotient.
